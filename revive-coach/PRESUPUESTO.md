# Presupuesto Desarrollo - Revive Coach IA

> Última actualización: Enero 2025
> Mercado de referencia: Madrid, España

---

## Stack Tecnológico

- **Frontend:** Next.js 14 + React 18
- **Backend/DB:** Supabase (PostgreSQL + Auth + Realtime)
- **UI:** Tailwind CSS + shadcn/ui + Framer Motion
- **Email:** Resend
- **Despliegue:** Vercel

- Los tiempos con IA asumen uso básico (Copilot, ChatGPT para snippets/debugging)

- Precios basados en mercado Madrid 2024-2025
